## Git Conventions

- Branch naming: `dos-XXX-short-description`
- Commit messages: Conventional Commits (`feat(DOS-XXX): description`, `fix(DOS-XXX): description`)
- Do NOT add "Co-Authored-By" lines to commit messages
- Always create a PR — do not push directly to main
